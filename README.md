1. Тема та мета проєкту
Сайт «CYBER VECTOR» — це сучасний веб-ресурс, який виконує роль інтерактивного портфоліо та освітнього блогу з векторної графіки та анімації. 
Мета проєкту: продемонструвати професійні навички дизайнера, поділитися навчальними матеріалами (фішками та уроками), зробити об'єктивний огляд профільний додатки та представити власний мультиплікаційний проєкт.

2. Використані технології та засоби (Стек)
Сайт розроблено з використанням "чистого" коду (Vanilla Web Technologies) без застосування важких фреймворків, що забезпечує його миттєве завантаження.
•	HTML5: Використано сучасну семантичну розмітку (теги <header>, <nav>, <footer>). Також застосовано нативні інтерактивні елементи <details> та <summary> для створення "акордеону" (ефект розгортання тексту) без використання JavaScript. Для інтеграції відеоконтенту використано тег <iframe>.
•	CSS3: Для побудови структури та вирівнювання елементів використано сучасні модулі Flexbox та CSS Grid. Наприклад, для галереї концепт-артів реалізовано горизонтальну прокрутку (flex-wrap: nowrap).
•	CSS Змінні (Custom Properties): Використано для оптимізації коду та легкого керування кольоровою палітрою сайту (наприклад, головним неоновим кольором).
•	Хостинг: Проєкт розміщено на безкоштовному хмарному сервісі GitHub Pages що забезпечує стабільний доступ до сайту через інтернет.

3. Особливості дизайну (UI/UX)
•	Темна тема (Dark UI): Сайт виконаний у глибоких темних відтінках із яскравими неоновими акцентами. Це знижує напругу на очі користувача та відповідає сучасним трендам у сфері діджитал-арту та кібер-стилістики.
•	Інтерактивність: Додано плавні мікроанімації за допомогою CSS-властивостей transition та transform. При наведенні курсору на картки, кнопки або мініатюри картинок вони плавно збільшуються та отримують неонове світіння (box-shadow), що робить інтерфейс "живим" та чуйним.
•	Адаптивність (Responsive Design): Блоки автоматично підлаштовуються під розмір екрану (наприклад, відео з YouTube зберігає ідеальні пропорції 16:9 завдяки властивості aspect-ratio).

4. Структура ресурсу (Сторінки)
1.	Головна: Вступний екран та знайомство з проєктом.
2.	Додатки: Аналітика мобільних додатків для малювання з детальним розбором переваг (+) та недоліків (-), а також зведена порівняльна таблиця.
3.	Уроки та Фішки: Навчальний розділ, де професійні секрети сховані під спойлерами для економії простору.
4.	Роботи: Галерея векторних ілюстрацій та окремий блок "Анімаційний проєкт", який містить вбудований плеєр з мультсеріалом та галерею робочих сцен
